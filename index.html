The First International Workshop on Datacenter Networking and Applications (DNA)

in conjunction with the 23rd International Conference on Algorithms and Architectures for Parallel Processing (ICA3PP 2023), Tianjin, China, 20-22 October 2023

Call for Papers

Datacenter has been built around the world to deliver various services to customers, including machine learning, database query. Datacenter networking and applications are two major research directions in datacenters. This workshop provides a forum for academic researchers and industry practitioners to exchange the most recent progress in methods and applications in datacenter networking and applications.

The workshop solicits submissions from areas relating to datacenter networking, such as networking topology, congestion control, load balance, etc., and datacenter applications, such as machine learning, database, serverless computing, etc. The interested topics include but not limited to:

  • Datacenter networking architectures/topologies
  • Datacenter networking transports
  • Flow scheduling/load balancing in datacenter networking
  • Programmable switches and SmartNICs
  • Datacenter networking monitoring and diagnostic
  • Big data analytical applications in datacenters
  • Machine learning/LLM applications in datacenters
  • Hardware offloading applications in datacenters
  • Privacy-preserving applications/federated learning in datacenters
  • Database in datacenters
  • Serverless computing
  • Cloud computing

Submitted manuscripts must be prepared according to IEEE Computer Society Proceedings Format (double column, 10pt font, letter paper) and submitted in PDF file format. https://www.ieee.org/conferences/publishing/templates.htmlThe manuscript should be no longer than 6 pages and at least 4 pages. Submitted manuscripts must not contain previously published material or be under consideration for publication in another conference or journal.
